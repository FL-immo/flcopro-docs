# Gestion des copropriétés

La section **Gestion des copropriétés** regroupe tous les outils nécessaires pour créer, consulter et gérer vos copropriétés. Elle est organisée en trois pages principales pour vous offrir une navigation optimale.

---

## Vue d'ensemble

La [vue d'ensemble](condominiums-overview.md) est votre page d'accueil pour accéder rapidement à vos copropriétés favorites.

**Fonctionnalités** :
- Affichage des **6 copropriétés épinglées** sous forme de cartes visuelles.
- Accès rapide aux fiches détaillées en un clic.
- Possibilité de **désépingler** directement depuis cette page.

> Épinglez vos copropriétés les plus consultées pour y accéder instantanément depuis la vue d'ensemble.
>
{style="tip"}

[Consulter la vue d'ensemble](condominiums-overview.md)

---

## Liste des copropriétés

La [liste des copropriétés](condominiums-list.md) centralise l'ensemble de vos copropriétés avec des outils de recherche et filtrage avancés.

**Fonctionnalités** :
- **Recherche multi-critères** : Filtrez par nom, ville ou code.
- **Cartes visuelles** : Chaque copropriété est présentée avec photo, adresse et informations clés.
- **Épinglage rapide** : Marquez vos favoris directement depuis les cartes (maximum 6).
- **Accès aux détails** : Cliquez sur une carte pour ouvrir la fiche complète.

> Utilisez les filtres combinés pour retrouver rapidement une copropriété parmi des centaines d'entrées.
>
{style="tip"}

[Accéder à la liste](condominiums-list.md)

---

## Détails d'une copropriété

La page [détail](condominiums-detail.md) affiche toutes les informations d'une copropriété et permet de la gérer.

**Informations affichées** :
- Nom, code et photo de la copropriété
- Adresse(s) complète(s)
- Date de création
- Liste de tous les rapports associés

**Actions disponibles** :
- **Modifier la copropriété** : Mettre à jour les informations, l'adresse ou la photo.
- **Créer un rapport** : Démarrer un nouveau rapport de visite directement depuis la fiche.
- **Consulter les rapports** : Accéder à tous les rapports liés à cette copropriété.
- **Supprimer la copropriété** : Suppression définitive (avec confirmation).

[En savoir plus sur les détails](condominiums-detail.md)

---

## Créer une copropriété

La page [Ajouter une copropriété](condominiums-create.md) permet de créer de nouvelles fiches via un formulaire guidé.

**Informations à renseigner** :
- **Nom** : Nom officiel de la copropriété *(obligatoire)*
- **Code** : Identifiant unique (suggestion automatique modifiable) *(obligatoire)*
- **Date de création** : Date ou année de création *(obligatoire)*
- **Photo** : Image de la copropriété (format paysage recommandé) *(optionnel)*
- **Adresse(s)** : Une ou plusieurs rues avec code postal, ville et pays *(obligatoire)*

> Le code est auto-suggéré en incrémentant le dernier code enregistré, mais reste entièrement modifiable selon vos conventions.
>
{style="note"}

[Ajouter une nouvelle copropriété](condominiums-create.md)

---

## Modifier une copropriété

La modification d'une copropriété se fait depuis la page [détail](condominiums-detail.md) via le bouton **Modifier**.

**Éléments modifiables** :
- Nom et code de la copropriété
- Date de création
- Photo (remplacement ou suppression)
- Adresse(s) : modification, ajout ou suppression de rues

> Lors de la modification, vous pouvez ajouter ou supprimer des adresses si la copropriété possède plusieurs entrées ou bâtiments.
>
{style="tip"}

[En savoir plus sur la modification](condominiums-edit.md)

## Flux de travail typique

<tabs>
<tab title="Créer et épingler une copropriété">

1. Cliquer sur **Copropriétés** → **Nouvelle copropriété**.
2. Remplir le formulaire (nom, code, date, adresse).
3. Ajouter une photo (recommandé).
4. Cliquer sur **Créer**.
5. Depuis la [liste](condominiums-list.md), cliquer sur l'icône bookmark pour épingler.
6. La copropriété apparaît maintenant dans la [vue d'ensemble](condominiums-overview.md).

</tab>
<tab title="Rechercher et modifier une copropriété">

1. Accéder à **Copropriétés** (liste).
2. Utiliser les filtres pour trouver la copropriété (nom, ville ou code).
3. Cliquer sur la carte pour ouvrir le détail.
4. Cliquer sur **Modifier**.
5. Mettre à jour les informations souhaitées.
6. Cliquer sur **Enregistrer**.

</tab>
<tab title="Créer un rapport depuis une copropriété">

1. Ouvrir le détail de la copropriété.
2. Cliquer sur **Créer un rapport**.
3. Sélectionner le type de rapport (Visite, Index, Maintenance).
4. Remplir le formulaire de rapport.
5. Enregistrer ou finaliser le rapport.

</tab>
</tabs>

## Bonnes pratiques

| Pratique | Recommandation                                                                           |
|----------|------------------------------------------------------------------------------------------|
| **Photos** | Utilisez des photos en format paysage pour la photo de la copropriété.                   |
| **Codes** | Adoptez une convention de codification cohérente (ex. `CDM-XXX`, `BXXX`).                |
| **Adresses multiples** | Ajoutez toutes les rues si la copropriété possède plusieurs accès.                       |
| **Épinglage** | Épinglez uniquement vos 6 copropriétés les plus consultées pour une navigation optimale. |
| **Suppression** | Vérifiez qu'aucun rapport important n'est lié avant de supprimer une copropriété.        |
