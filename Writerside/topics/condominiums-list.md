# Copropriétés — Liste

La page **Liste des copropriétés** centralise l'ensemble des copropriétés enregistrées et offre des outils de recherche, de filtrage et d'accès rapide aux fiches détaillées.

![condominium-list.png](condominium-list.png)

---

## Raccourci d'utilisation

1. Ouvrir **Copropriétés** (menu principal → Copropriétés).
2. Utiliser les **filtres** (Nom, Ville, Code) pour affiner la recherche.
3. **Cliquer sur une carte** pour accéder au [détail de la copropriété](condominiums-detail.md).
4. Utiliser l'**icône bookmark** pour épingler/désépingler une copropriété dans la [vue d'ensemble](condominiums-overview.md).
5. Cliquer sur **Nouvelle Copropriété** pour [ajouter une copropriété](condominiums-create.md).

---

## Filtres de recherche

Trois champs de filtrage permettent d'affiner la liste affichée :

| Filtre    | Description                                                                    | Exemple                         |
|-----------|--------------------------------------------------------------------------------|---------------------------------|
| **Nom**   | Recherche par nom de copropriété (recherche partielle, insensible à la casse). | `Résidence`, `Parc`, `Marivaux` |
| **Ville** | Filtre par ville (recherche partielle).                                        | `Paris`, `Lyon`, `Belfort`      |
| **Code**  | Recherche par code unique de copropriété.                                      | `CDM-001`, `B0018`, `9`         |

> Les filtres sont cumulatifs : vous pouvez combiner plusieurs critères pour affiner votre recherche.
>
{style="tip"}

## Cartes des copropriétés

Chaque copropriété est représentée par une **carte visuelle** contenant :

| Élément            | Description                                                                        |
|--------------------|------------------------------------------------------------------------------------|
| **Photo**          | Image de la copropriété.                                                           |
| **Nom**            | Nom complet de la copropriété (ex. *Le marivaux*, *Résidence du Parc Montrosier*). |
| **Adresse**        | Rue, code postal et ville (première adresse enregistrée si plusieurs rues).        |
| **Code**           | Identifiant unique de la copropriété (ex. `CDM-001`, `B0018`).                     |
| **Créée**          | Année de création de la copropriété (ex. *1970*, *2008*).                          |
| **Icône bookmark** | Épingler/désépingler la copropriété (plein = épinglée, vide = non épinglée).       |


---

## Interactions disponibles

### Cliquer sur une carte

Ouvre la [page de détail](condominiums-detail.md) de la copropriété correspondante, permettant de consulter toutes les informations, modifier la fiche ou créer un rapport.

---

## Exemples d'utilisation

<tabs>
<tab title="Rechercher une copropriété par nom">

1. Saisir le nom (ou une partie) dans le champ **Nom**.
2. La liste se met à jour automatiquement.
3. Cliquer sur la carte correspondante pour ouvrir le détail.

**Exemple** : saisir `Résidence` affiche toutes les copropriétés contenant ce mot dans leur nom.

</tab>
<tab title="Filtrer par ville">

1. Saisir le nom de la ville dans le champ **Ville**.
2. Seules les copropriétés de cette ville s'affichent.

**Exemple** : `Paris` affiche uniquement les copropriétés parisiennes.

</tab>
<tab title="Recherche par code">

1. Saisir le code complet ou partiel dans le champ **Code**.
2. La liste filtre immédiatement les résultats.

**Exemple** : `CDM-001` affiche la copropriété avec ce code exact.

</tab>
<tab title="Épingler une copropriété">

1. Repérer la copropriété souhaitée dans la liste.
2. Cliquer sur l'**icône bookmark** (en haut à droite de la carte).
3. L'icône se remplit → la copropriété apparaît désormais dans la [vue d'ensemble](condominiums-overview.md).
4. **Note :** Vous ne pouvez épingler que **6 copropriétés maximum**. Si vous tentez d'épingler plus, un message d'alerte s'affichera.

</tab>
</tabs>
