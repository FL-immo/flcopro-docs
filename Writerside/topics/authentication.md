# Authentification

Pour accéder à **%product%**, vous devez d’abord vous connecter à la plateforme.

## Prérequis

Avant de commencer, assurez-vous de disposer :

- De vos identifiants de connexion (fournis par votre administrateur).
- De l’accès à l’URL de **[%product%](%product_url%)**.

## Accès à la page de connexion

Pour vous connecter, rendez-vous sur l’interface d’authentification de **%product%**.  
Si vous accédez à une page nécessitant une session active, vous serez automatiquement redirigé vers cette page.

L’écran de connexion se présente comme suit :

![Écran de connexion](login-page.png)

Vous y trouverez deux champs :

- **Email** : votre adresse professionnelle (ex. : `p.nom@prevot-immobilier.com`)
- **Mot de passe** : le mot de passe fourni par votre administrateur

Une fois les champs renseignés, cliquez sur **Se connecter**.

## Gestion du mot de passe

### Mot de passe oublié
En cas d’oubli de votre mot de passe, contactez votre administrateur pour en demander la réinitialisation.

### Modifier votre mot de passe
Une fois connecté, vous pouvez mettre à jour votre mot de passe depuis les paramètres de votre [compte utilisateur](profile.md#password).

## Messages d’erreur

> Si la connexion échoue, le message suivant peut s'afficher : `Identifiants invalides ou erreur serveur`, dans ce cas, vérifiez vos informations de connexion et réessayez.
