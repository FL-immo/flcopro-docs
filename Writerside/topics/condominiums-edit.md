# Copropriétés — Modifier

La page **Modifier la copropriété** permet de mettre à jour les informations d'une copropriété existante : nom, code, date de création, photo et adresses.

---

## Raccourci d'utilisation

1. Accéder à la page depuis le [détail de la copropriété](condominiums-detail.md) → **Modifier les informations**.
2. Modifier les champs souhaités (nom, code, date, photo).
3. Ajouter, modifier ou supprimer des adresses via **Ajouter une rue**.
4. Cliquer sur **Enregistrer** pour sauvegarder les modifications.

---

## Champs modifiables

| Champ | Description |
|-------|-------------|
| **Nom** | Nom de la copropriété. |
| **Code** | Identifiant unique (ex. `B0018`, `CDM-001`). |
| **Date de création** | Date ou année de construction (format `jj/mm/aaaa`). |
| **Photo** | Image de la copropriété (format paysage recommandé). |

---

## Adresses

La copropriété peut avoir plusieurs adresses (rues).

| Champ | Description |
|-------|-------------|
| **Rue** | Numéro et nom de la rue. |
| **Code postal** | Code postal. |
| **Ville** | Nom de la ville. |
| **Pays** | Pays (par défaut : France). |

### Gestion des adresses

- **Ajouter une rue** : cliquer sur le bouton pour ajouter un nouveau bloc d'adresse.
- **Supprimer une adresse** : cliquer sur l'icône de suppression à droite du bloc.

> Vous pouvez enregistrer plusieurs adresses pour une même copropriété (entrées multiples, accès secondaires).
>
{style="tip"}

---

## Actions disponibles

| Bouton | Description |
|--------|-------------|
| **Enregistrer** | Sauvegarde les modifications et retourne à la page de [détail](condominiums-detail.md). |
| **Annuler** | Annule les modifications et retourne à la page de détail sans sauvegarder. |
