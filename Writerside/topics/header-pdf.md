# Ajouter une page de garde à un PDF

L'outil de page de garde permet d'ajouter une page de couverture professionnelle au début de vos documents PDF existants.

## Accéder à l'outil page de garde

1. Dans le menu latéral, cliquez sur **Divers**
2. Cliquez sur **Page de garde**

## Interface de l'outil

L'interface se présente sous forme d'un formulaire simple permettant de configurer votre page de garde.

## Télécharger le fichier PDF

### Méthode 1 : Glisser-déposer

1. Glissez votre fichier PDF depuis votre explorateur de fichiers
2. Déposez-le dans la zone de téléchargement

### Méthode 2 : Sélection manuelle

1. Cliquez sur la zone de téléchargement
2. Sélectionnez votre fichier PDF dans la boîte de dialogue

### Format accepté

- **Format** : PDF uniquement
- **Mode** : Liste (un seul fichier à la fois)

> Le fichier PDF sélectionné sera conservé tel quel, seule une page de garde sera ajoutée au début.

## Configurer la page de garde

### Titre de la page de garde

Ce champ permet de définir le titre qui apparaîtra sur la page de garde.

#### Champ éditable avec suggestions

Le champ **Titre** propose :
- Une saisie libre
- Des suggestions basées sur des titres couramment utilisés

#### Utilisation

1. Cliquez dans le champ **Titre de la page de garde**
2. Commencez à taper votre titre
3. Sélectionnez une suggestion dans le menu déroulant, ou
4. Continuez à taper votre titre personnalisé

> **Astuce** : Les suggestions apparaissent automatiquement en fonction des titres déjà utilisés dans l'application.

#### Exemples de titres

- "Rapport de visite technique"
- "Compte rendu d'assemblée générale"
- "Procès-verbal de réunion"
- "État des lieux"
- "Rapport d'expertise"

### Sélectionner la copropriété

1. Cliquez sur le menu déroulant **Copropriété**
2. Sélectionnez la copropriété concernée dans la liste

> La copropriété apparaîtra sur la page de garde pour identifier clairement le document.

### Sélectionner la date

1. Cliquez sur le champ **Date**
2. Sélectionnez la date dans le calendrier, ou
3. Saisissez la date manuellement au format JJ/MM/AAAA

> La date permet de dater officiellement le document.

## Générer le PDF avec page de garde

### Vérifier les informations

Avant de générer, assurez-vous que :
- ✅ Un fichier PDF est bien chargé
- ✅ Le titre est renseigné
- ✅ Une copropriété est sélectionnée
- ✅ Une date est saisie

### Lancer la génération

1. Cliquez sur **Générer le PDF avec page de garde**

### Traitement en cours

Pendant le traitement :
- Le bouton affiche **"Traitement en cours..."**
- Le bouton est désactivé
- Patientez quelques instants

> **Important** : Ne fermez pas la page pendant le traitement.

### Résultat

Une fois le traitement terminé :
- Le nouveau PDF avec la page de garde est automatiquement téléchargé
- Le fichier téléchargé contient :
  - La page de garde générée (première page)
  - Toutes les pages de votre PDF original (pages suivantes)

## Composition de la page de garde

La page de garde générée contient les éléments suivants :

### En-tête

- **Logo** : Logo de l'application (si configuré)
- **Nom de l'application** : FL Copro

### Corps

- **Titre principal** : Le titre que vous avez saisi
- **Copropriété** : Nom de la copropriété sélectionnée
- **Adresse** : Adresse complète de la copropriété
- **Date** : Date sélectionnée

### Pied de page

- **Coordonnées** : Coordonnées du cabinet ou du gestionnaire
- **Mentions légales** : Mentions légales éventuelles

> Le design de la page de garde est professionnel et adapté aux documents officiels.

## Cas d'usage

### Documents adaptés

- Rapports de visite
- Procès-verbaux
- Comptes rendus de réunion
- États des lieux
- Expertises
- Devis et factures
- Contrats
- Courriers officiels

### Avantages

- **Professionnalisme** : Présentation soignée et uniforme
- **Identification** : Copropriété clairement identifiée
- **Traçabilité** : Date du document visible
- **Archivage** : Facilite le classement des documents

## Bonnes pratiques

### Choix du titre

- Soyez **précis et descriptif**
- Évitez les titres trop longs
- Utilisez les **conventions** de votre organisation
- Exemple : Préférez "Rapport de visite technique - Toiture" à "Rapport"

### Sélection de la copropriété

- Vérifiez que vous avez sélectionné la **bonne copropriété**
- En cas de doute, vérifiez l'adresse affichée

### Date du document

- Utilisez la **date de création** du document original
- Ou la **date de finalisation** du rapport
- Soyez cohérent dans toute votre documentation

### Nommage du fichier

Après téléchargement, renommez votre fichier de manière claire :
- Format suggéré : `[Date]_[Type]_[Copropriété].pdf`
- Exemple : `2026-01-06_Rapport-visite_Residence-Parc.pdf`

## Workflow recommandé

1. **Préparez** votre document PDF original
2. **Accédez** à l'outil Page de garde
3. **Chargez** votre PDF
4. **Remplissez** les informations de la page de garde
5. **Générez** le PDF final
6. **Téléchargez** le résultat
7. **Renommez** le fichier de manière appropriée
8. **Archivez** ou envoyez le document

## Différences avec la création de rapport

### Outil Page de garde

- Ajoute une page de couverture à un PDF **existant**
- Rapide et simple
- Pour des documents **déjà créés** (Word, Excel, etc.)
- Idéal pour uniformiser les documents externes

### Création de rapport

- Crée un rapport **complet de A à Z**
- Formulaires structurés avec de nombreux champs
- Génère le PDF automatiquement
- Pour les rapports de visite, sinistres, etc.

> Utilisez l'outil **Page de garde** pour vos documents déjà prêts, et la **Création de rapport** pour générer des rapports structurés depuis l'application.

## Dépannage

### Le PDF ne se génère pas

**Causes possibles :**
- Fichier PDF corrompu
- Fichier trop volumineux
- Connexion internet interrompue

**Solutions :**
- Vérifiez que votre PDF s'ouvre correctement
- Essayez avec un autre fichier PDF
- Actualisez la page et réessayez

### Le téléchargement ne démarre pas

**Solutions :**
- Vérifiez les paramètres de téléchargement de votre navigateur
- Autorisez les téléchargements depuis l'application
- Désactivez les bloqueurs de pop-up
- Essayez avec un autre navigateur

### Les informations sont incorrectes sur la page de garde

**Solutions :**
- Vérifiez que vous avez sélectionné la bonne copropriété
- Vérifiez le titre saisi
- Vérifiez la date sélectionnée
- Régénérez le PDF après correction
